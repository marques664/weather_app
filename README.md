# Weather App

Mobile application developed with Flutter that provides weather information based on the user's location or a searched city.

This repository contains two different versions of the application, showing the evolution of the project and different approaches to user interaction.

## Technologies
- Flutter
- Dart
- REST API
- HTTP requests
- Geolocation

## Project Structure

### Version 1
Search weather by typing the city name with autocomplete suggestions.

Features:
- City search with autocomplete
- Weather data visualization
- Simple interface

Folder: weather_app/


### Version 2
More minimalistic version that prioritizes user convenience by using GPS location, while still allowing manual city search.

Features:
- Automatic weather detection using GPS
- Manual city search option
- Cleaner interface
- Improved usability

Folder: weather_app_v2/


## Purpose
This project was created to practice:

- API integration
- handling asynchronous data
- geolocation services
- improving user experience
- structuring Flutter applications
